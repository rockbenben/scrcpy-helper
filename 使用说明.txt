========================================
  scrcpy 投屏助手 · 使用说明
========================================

这是一个把安卓手机投屏到 Windows 电脑的小工具，基于开源软件 scrcpy。
可以用电脑鼠标键盘操控手机、无线投屏、录屏、把手机当摄像头，
还能把某个 App 单独放到电脑上的独立窗口里运行。完全免费、开源、无广告。


----------------------------------------
一、怎么用（三步）
----------------------------------------
1. 把压缩包【解压】到任意文件夹（不要直接在压缩包里运行）。

2. 手机开启「USB 调试」：
   设置 > 关于手机 > 连续点 7 次「版本号」，返回设置即可看到
   「开发者选项」，进去打开「USB 调试」。

3. 双击运行【投屏助手-双击运行.bat】，在弹出的窗口里点按钮即可使用。

提示：首次用数据线连接手机时，手机会弹窗，请点「允许 USB 调试」。
窗口顶部会显示「已连接 / 未连接」，连上后还会显示手机型号和安卓版本
（如「Android 13 · Redmi Note 11」），一眼就知道哪些功能能用，连上后再点按钮。


----------------------------------------
二、功能一览
----------------------------------------
  有线投屏      数据线连手机，最简单、最稳定、延迟最低
  无线投屏      插一次线自动配置即可拔线；Android 11+ 用配对码免插线，或直接输 IP 连接作保底（不限版本）
  手机当摄像头   把手机摄像头变成电脑摄像头（可选前/后置、横/竖屏、分辨率、补光灯，需 Android 12+；会记住上次选择。
                手机亮屏被系统抢走摄像头会自动重连；选的分辨率打不开会自动降一档重试）
  录制屏幕      边投屏边录，存成 mp4 / mkv（文件名自动带时间，不会覆盖上一次）
  独立窗口      在电脑上单开一块屏跑某个 App，手机还能照常用；可选竖屏/横屏/自定义比例
                与「固定方向」（缓解最大化乱转），需 Android 11+
  设备管理      点状态药丸或「设备」：连接/投屏分开、停止指定设备、设为默认、重命名、断开、
                忘记、输入 IP 连接，还能「全部投屏」/ 多选同时投多台（连过的设备会自动记住）

注：摄像头 / 独立窗口对系统版本有要求，手机版本不够时点了会有提示，不会瞎忙。
独立窗口选 App 时，除了内置常用 App，还能用「更多应用…」从手机已装列表里直接挑
（可搜索、带中文名，不必查包名）；挑完可一键加进自己的常用列表，下次直接选，
也能在「管理我的常用应用…」里手动增删，扩成你自己的快捷清单。

直接点按钮就能用，全部采用合理的默认设置，新手无需任何配置。

· 投屏时，把文件直接拖进投屏窗口即可传到手机，拖入 APK 可一键安装。
· 点主界面右下角「快捷键」可查看常用快捷键（全屏、息屏、旋转、复制粘贴等）。
· 关闭助手窗口（×）会一并停止投屏（正在录屏会先问你确认，确认后会让录制正常收尾，录好的文件完整可播，后台录制也一样）。
· 想让投屏继续、只把助手收起来，请点最小化（–）。
· 助手同一时间只开一个：已经开着时再双击【投屏助手-双击运行.bat】，只会把已有窗口激活到最前，不会又开一个。


----------------------------------------
三、想自定义？点窗口右下角「设置」
----------------------------------------
设置里左侧按分类切换，右侧是对应的可调项，改完自动记忆，下次沿用：

  常用    清晰度、传声音、保持唤醒、投屏关屏、无线自动重连/断开（最常调的都在这）
  画面    帧率、码率、视频编码（H.264/H.265/AV1）、裁剪画面（只投一块区域）
  声音    声音来源（外放/麦克风）、音频编码
  控制    键盘/鼠标模式、只看不控、关窗顺手息屏、显示触摸点、手柄
  窗口    全屏、窗口置顶、无边框
  独立窗口  分辨率（可自定义）、界面缩放、隐藏系统栏
  录制    保存格式、录制时长上限、后台录制
  通用    自动刷新状态、其它命令行参数（高级，看不懂留空）、
          自定义 adb / scrcpy 路径（留空＝用自带的；想用电脑里别处装的才填，保存即时生效）

不确定某项是做什么的，把鼠标停在它上面会有中文说明。
看不懂的保持默认即可，不影响正常使用。
改乱了想还原？设置窗口左下角有「恢复默认」按钮，一键复位。


----------------------------------------
四、连接与省电（下面这些都在「设置 > 常用」里）
----------------------------------------
• 无线投屏「点一下就开」：已经连着手机就直接投，
  只有没检测到手机时，才会让你选「插线连接」或「配对码连接」。

• 为什么要插一下线？插线是为了借这条已授权的通道，打开手机的无线调试开关
  并完成一次授权；开关打开后即可拔线、纯走 WiFi。注意：手机重启后这个开关会
  关掉，需要重新插一次线（用下面的配对码方式则不必）。

• 免插线连接（手机为 Android 11 及以上）：在「无线投屏 > 没检测到手机」时选
  「用配对码连接」。先在手机开「开发者选项 > 无线调试」，点「使用配对码配对设备」，
  把那个弹窗里的【IP 地址和端口】和【6 位配对码】照抄进来即可——两项就并排在
  同一个弹窗里，连接端口会自动识别，不用你再去看别处。
  配对码会过期，慢了就重新生成；配一次长期有效，以后重连也不再需要配对码
  （重启后用「自动重连」连回即可）。

• 输入 IP 直接连接（保底连法，不限系统版本）：在「无线投屏 > 没检测到手机」时选
  「输入 IP 直接连接」，填手机 IP（端口一般是 5555）即可，不需要配对码。新旧手机都适用，
  重连用过的设备也方便。前提是手机已开启网络 adb：Android 11+ 在「无线调试」里就有；
  更早的系统或从没开过的，先用「插数据线连接」插一次线完成切换，之后就能纯 IP 连/重连了。

• 多台设备怎么管理 / 同时投屏？点右上角状态药丸（或右下角「设备」）打开「设备管理」，
  按钮分「投屏 / 管理」两组：
  - 投屏组：「连接」只负责把没连的设备连上；连上后再点「投屏」开镜像窗口（两步分开、更清晰）。
    双击某行＝顺手帮你做下一步（没连就连、已连就投）。
  - 想同时投多台：按住 Ctrl 多选后点「投屏」，或直接「全部投屏」，每台各开一个窗口、互不干扰
    （窗口标题就是设备名，便于区分）；正在投的设备会标「▶ 投屏中」，「停止投屏」只关选中那台。
  - 管理组：「设为默认」（多设备时默认投它）、「断开」、「重命名」（给设备起个好认的名字，
    如「客厅电视」）、「忘记」。
  这样不必再去关手机的无线调试开关，也不会只记得最后一台。
  （主界面的「有线/无线投屏」按钮若同时连了多台，会让你先挑一台；设过「默认」的直接用默认那台。）

• 老是中途断开？勾「无线掉线后自动重连」，断了会自动连回来。

• 设备暂时没连、但以前无线连过？直接点「无线投屏」即可（录屏、摄像头、独立窗口、有线投屏
  也一样）——助手会先自动连回「记住的、当前在线」的设备再开始，不必每次都重新选连接方式。
  连接时会弹一个「正在连接已记住的设备…」小提示；只有都连不上，才会让你选连接方式。
  （记住的设备若当前不在线，会被快速跳过，不会卡住等待。）

• 想更省手机电（注意：只有正在投屏时才真费电）：
  · 勾「投屏时关闭手机屏幕」——最省电，还能防别人偷看；
  · 或取消「保持手机唤醒」——让手机能正常休眠。

• 关掉助手后，手机还连着是正常的，几乎不耗电，重开就能直接用。
  如果想关助手时一并断开，勾「关闭助手时断开无线连接」即可。


----------------------------------------
五、常见问题
----------------------------------------
• 双击后窗口一闪而过 / 打不开：
  右键【投屏助手-双击运行.bat】>「以管理员身份运行」试试；
  若被杀毒软件或 Windows 拦截，请选择「更多信息 > 仍要运行」。

• 提示找不到设备：
  换一根原装数据线、直插电脑 USB 口（别用扩展坞），
  并确认手机已打开「USB 调试」。

• 无线连不上：
  确认手机和电脑连的是同一个 WiFi。
  用配对码时还要确认手机已打开「无线调试」并保持那个页面别退出。
  若助手提示「没能自动识别连接端口」，按提示填手机无线调试主界面
  那个「IP 地址和端口」里冒号后面的端口号即可。

• 投屏出来的手机画面发虚模糊（指镜像的手机屏幕，不是助手界面）：
  右键 scrcpy.exe >「属性」>「兼容性」>「更改高 DPI 设置」>
  勾选「替代高 DPI 缩放行为」。
  （助手界面本身已适配高 DPI，在 150%/200% 缩放屏上是清晰的；若助手界面发虚，
   多为旧版本，更新到最新即可。）

• 摄像头打不开、命令行报「Camera configuration error」或「Could not retrieve device information」：
  本助手会在打开「手机当摄像头」时自动读取本机摄像头真正支持的分辨率，前置/后置支持的
  尺寸常不同，切换后下拉会自动刷新。个别超高档位（如 4000+ 宽）手机的视频编码器可能带不动，
  打不开时助手会自动换下一档尺寸重试，稳定后记住这档，下次直接用。
  （若你的 scrcpy 较旧、读不到支持列表，会退回「高/中/原始最高」档位，这时高的打不开就选「中」。）

• 手机一亮屏（来通知 / 人脸解锁），摄像头就断开：
  这是安卓的系统行为——亮屏时系统（如人脸解锁）优先级更高，会把摄像头从 scrcpy 手里抢走，
  scrcpy 挡不住。助手检测到这种中断后会自动重连摄像头（底部会有提示），无需手动重开；
  想少被打断，可在使用期间开勿扰、或关掉「抬起亮屏 / 人脸解锁」。

• 电脑输入法打的中文进不去投屏窗口（只能粘贴）：
  这是 scrcpy 与 Windows 输入法的已知限制——电脑输入法「组词」阶段的字 scrcpy 收不到。
  · 临时：在电脑里复制好文字，再到投屏窗口按 Ctrl+V 粘贴。
  · 彻底：给手机装「ADBKeyboard」并设为当前输入法；或在「设置 > 控制」把键盘模式
    选「游戏模式」，用手机自带输入法打拼音。

• 独立窗口里微信 / QQ 显示不全、字太大、有重影：
  那是应用把虚拟屏当成了平板。打开「独立窗口」时把「窗口比例」选「竖屏·手机版面」即可；
  若最大化 / 全屏后画面来回乱转，再勾上「固定方向」。

• 应用双开 / 分身在独立窗口里黑屏、点不到分身：
  这是安卓的限制——分身运行在另一个用户身份下，且部分应用拒绝在虚拟屏上显示，
  scrcpy 无法定向到分身。建议双开 / 分身改用普通「投屏」镜像手机主屏，在手机上打开分身使用。

注意：文件夹里的 scrcpy-helper.ps1、scrcpy.exe 等是程序文件，
平时不用打开，只要它们和 .bat 在同一个文件夹里即可。
